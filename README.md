# TypeScript 샘플
- URL : https://www.typescripttutorial.net/typescript-tutorial/typescript-array-type/
## 테스트를 위한 프로그램 버전
1. Node.js : v19.5.0
2. npm : 9.3.1
3. typescript 설치
```shell
$ npm install -g typescript
$ tsc --v
Version 5.1.6
```
4. ts-node 설치
```shell
$ npm install -g ts-node
```
